Detailed Description

Class to set up and run the estimation process in polytomous models

The main method is EMAlgorithm

Receives:

  1. A specific model to use -> [1, 3] that means 1PL, 2PL, or 3PL

A matrix containing the answers of examinees

  1. The dimension of the problem
  2. The epsilon (convergences difference) that the algoritm will use as a criterion to stop

Optional parameters: quadrature_technique: [GAUSSIAN_QUADRATURE, SOBOL_QUADRATURE] quadrature_points: Custom number of quadrature points cluster: Vector that contains the number of item for each dimension custom_initial_values_filename: Custom initial values filename

Then it sets up all the data needed to start the estimation process

void irtpp::polytomous::estimation::gaussian_quadrature ( )

gaussian_quadrature. By default max points to use is 40.

Number of quadrature points (G) is computed based on MAX_NUMBER_OF_QUADRATURE_POINTS and dimension of the problem, in this way

G will be in 1dimension = 40 —> 40^1 = 40 2dimension = 20 —> 20^2 = 400 3dimension = 10 —> 10^3 = 1000 > 4dimension = 5 —> 5^d

Finds the initial values for every parameter of the items to start the estimation. it is called if custom_initial_values_filename is none by default.
